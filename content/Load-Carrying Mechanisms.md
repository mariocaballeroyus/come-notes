## Shear Action






$$\begin{aligned}
\int_{\Omega} { \text{div}(\text{div}(\mathbf{M}))v \, \text{d}x } 
&=
- \int_{\Omega} { \text{div}(\mathbf{M}) \cdot \nabla v \, \text{d}x }
+ \int_{\partial\Omega} { (\text{div}(\mathbf{M})\mathbf{n})v \, \text{d}s }
\\ &=
\int_{\Omega} { \mathbf{M} : \nabla^2 v \, \text{d}x }
- \int_{\partial\Omega} { (\mathbf{M}\mathbf{n}) \cdot \nabla v \, \text{d}s }
+ \int_{\partial\Omega} { (\text{div}(\mathbf{M}) \cdot \mathbf{n})v \, \text{d}s }
\end{aligned}$$
